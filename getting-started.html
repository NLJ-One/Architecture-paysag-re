Bienvenue dans le Portfolio

Guide de démarrage et structure du site.

Pour commencer

Ce portfolio a été conçu pour être facile à utiliser et à personnaliser. Voici les étapes principales:

1. Remplacer les images

Placez vos images dans le dossier /images/:

  • ✓ hero-image.jpg
  • ✓ project-1.jpg, project-2.jpg, project-3.jpg
  • ✓ arch-1.jpg, arch-2.jpg
  • ✓ design-1.jpg, design-2.jpg
  • ✓ photo-1.jpg, photo-2.jpg
  • ✓ about-portrait.jpg

2. Mettre à jour vos informations

Modifier dans les fichiers HTML:

  • ✓ Email et téléphone dans contact.html
  • ✓ Bio et expertise dans about.html
  • ✓ Liens des réseaux sociaux dans le footer

3. Personnaliser les couleurs

Modifier les variables CSS dans css/style.css:

:root {
    --color-primary: #1a1a1a;
    --color-accent: #c4a77d;
}

Déployer le site

Vous pouvez héberger ce portfolio gratuitement sur:

  • ✓ Netlify
  • ✓ Vercel
  • ✓ GitHub Pages
  • ✓ Votre hébergeur FTP préféré

📞 Intégration du formulaire

Pour que le formulaire fonctionne réellement, utilisez un service comme:

  • ✓ Formspree
  • ✓ Netlify Forms
  • ✓ EmailJS

FAQ

Comment ajouter un nouveau projet?

Dupliquer un bloc <article class="project-item"> dans projects.html et adapter le contenu.

Comment changer la police?

Modifier --font-main et --font-display dans css/style.css.

Le site fonctionne sans internet?

Oui! C'est un site statique - pas de serveur nécessaire.
